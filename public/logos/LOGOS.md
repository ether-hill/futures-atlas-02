# /public/logos — sources

Monochrome single-path SVGs generated from the [simple-icons](https://simpleicons.org) dataset (CC0), fill=currentColor. Tools without an available brand icon (ChatGPT/OpenAI, Midjourney, Kling AI) render as typographic tiles by design — never a broken image.

- claude.svg — Claude (simple-icons, brand hex #D97757)
- anthropic.svg — Anthropic (simple-icons, brand hex #191919)
- huggingface.svg — Hugging Face (simple-icons, brand hex #FFD21E)
- nextjs.svg — Next.js (simple-icons, brand hex #000000)
- vercel.svg — Vercel (simple-icons, brand hex #000000)
- threejs.svg — Three.js (simple-icons, brand hex #000000)
- p5js.svg — p5.js (simple-icons, brand hex #ED225D)
- react.svg — React (simple-icons, brand hex #61DAFB)
- tailwindcss.svg — Tailwind CSS (simple-icons, brand hex #06B6D4)
- d3.svg — D3 (simple-icons, brand hex #F9A03C)
